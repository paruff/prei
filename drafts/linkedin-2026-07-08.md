HUD, USDA, Dallas County, and ATTOM preforeclosure data — all in one pipeline.

prei v0.4.0 adds six new discovery source integrations built on a shared model layer. HUD REO properties now come from the real ArcGIS Hub endpoint (not a placeholder). USDA REO parses the government's fixed-width TXT feed. Dallas County NTS notices run through Playwright-based scraping. And the existing ATTOM adapter now supports ZIP-based preforeclosure search.

The stack is Python 3.14 + Django 6.0. Reportlab 5 is unblocked (xhtml2pdf replaced by Playwright for PDF generation). Every source flows into the same screening and pipeline workflow.

Open source at github.com/paruff/prei. Built for buy-and-hold real estate investors who want one tool, not six bookmarks.
