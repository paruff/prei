ℹ How It Works

Click Analyze State to run a live analysis:

  1. Discover places: Queries Census for the top 10 places by population in the selected state.
  2. Pull metrics: For each place, fetches 5-year population growth, median income growth, and housing demand index from Census ACS.
  3. Add employment: Pulls 5-year state-level employment growth from FRED (Federal Reserve) — same value for all places in the state.
  4. Score & rank: Computes a composite score and stores results for future reference.
Timing: Takes ~20–60 seconds depending on state size. Up to 21 sequential Census/FRED API calls.
📊 Data Sources & Output
Census ACS (5-year):
  • Population & population growth (5yr)
  • Median income & income growth (5yr)
  • Housing demand index (derived)
FRED (Federal Reserve):
  • State-level employment growth (5yr) via nonfarm payrolls
  • Same value for all places in the state
Results table includes:
  • Rank (by composite score)
  • City, State, Population (formatted)
  • Pop Growth %, Emp Growth %*, Income Growth %
  • Housing Demand Index (0–100)
  • Composite Score (higher = stronger growth profile)
  • Action: Direct link to VRM foreclosures in that state
